📊 Recruitment Analytics
This project performs a comprehensive analysis of recruitment data to uncover hiring trends, recruiter performance, and placement effectiveness.
🔍 Key Analyses Performed

Date Formatting – Standardized application dates to dd/mm/yyyy
Interview Sheet Lookup – Mapped TL Names using Lookup Tables
Round-wise Funnel – Candidate-level and recruiter-level counts per round
Rejection Analysis – Identified recruiters with highest rejection count & rejection %
Recruiter Performance – Ranked recruiters by placement rate; flagged high-application but low-placement recruiters
Source Analysis – Evaluated whether hiring source impacts placement outcomes
Looker Studio Dashboard – KPIs, funnel charts, source performance, and top recruiter highlights

💡 Actionable Insights
Top 2–3 data-driven recommendations to improve hiring efficiency and reduce rejection rates.
🛠 Tools Used
Excel · Looker Studio (Data Studio) · Google Docs
